Featured Project

Valor (github.com/tomcounsell/ai) is an open-source autonomous development system — an AI coworker that executes full software development lifecycles independently. Unlike AI coding assistants that generate code on demand, Valor is a persistent agent with its own identity, machine, memory, and professional judgment. It communicates via Telegram as a regular user account, receives work assignments, and autonomously plans, builds, tests, patches, reviews, documents, and merges code — only escalating when genuinely blocked. The system is built on the Claude Agent SDK with Python (~4M lines), actively developed since September 2020 with 2,300+ total commits. Valor itself is the primary contributor (1,970 commits vs 351 from Tom Counsell), demonstrating that the system meaningfully develops its own codebase. Architecture highlights: (1) Full SDLC pipeline with 10+ composable stages (Plan → Critique → Build → Test → Patch → PR Review → Docs → Merge → Deploy), each implemented as a distinct skill. (2) Multi-agent orchestration via the P-Thread pattern — spawning parallel sub-agents, chaining sequential work, fusing results, and running nested agent hierarchies. (3) Subconscious memory system using bloom filter-based recall with importance-weighted records, decay over time, and outcome-based reinforcement — memories surface automatically during work without explicit retrieval. (4) Self-healing infrastructure with 5-level crash escalation (restart → kill stale → clear locks → revert commit → alert human), a bridge watchdog, and crash tracking with git commit correlation. (5) Daily autonomous maintenance ("Reflections") — a 14-unit pipeline running at 6 AM that handles legacy code scanning, log review, documentation audits, behavioral learning, and auto-generated daily reports filed as GitHub issues. (6) 35+ skills/commands, 30+ local Python tools (browser automation, image analysis, SMS reading for 2FA, memory search, code impact analysis, etc.), and MCP integrations with Sentry, Notion, GitHub, and Google Workspace. (7) Autoexperiment framework that iteratively optimizes its own prompts through measured experiments. (8) Session isolation with per-work-item git worktrees and task lists enabling true parallel work. Tom's clients deploy Valor as a team member on their engineering teams — it receives assignments, ships code, and reports outcomes. The system embodies a "build the system that builds the system" philosophy, with a stated goal of zero-touch autonomous delivery where agents ship verified results while the human sleeps.
